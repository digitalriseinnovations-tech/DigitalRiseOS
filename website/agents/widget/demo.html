Pass a full config via data-config (base64 JSON) to override KB and greeting without an API call.

// Build your config object
const config = {
  name: "Your Business",
  agentName: "Ava",
  color: "#6366F1",
  greeting: "Hi! How can I help?",
  services: ["Get a Quote", "Book a Visit"],
  qualQs: ["What are you looking for?"],
  kb: {
    hours: { answer: "Mon–Fri 9am–5pm", keywords: ["hours", "open"] }
  }
};

// Base64 encode and add to script tag
const encoded = btoa(JSON.stringify(config));

<script src="…/widget.js"
  data-biz="your-biz-slug"
  data-config="{encoded}"></script>
How to install
3-step installation
Works on any website — WordPress, Squarespace, Wix, or custom HTML.
1
Copy the embed snippet above

Choose your business from the selector and click "Copy" on the code block.

2
Paste before </body> on your website

In WordPress: Appearance → Theme Editor → footer.php. In Squarespace/Wix: Site settings → Custom code → Footer.

3
The widget loads automatically

No restart needed. Refresh your website and the chat bubble appears in the corner.
